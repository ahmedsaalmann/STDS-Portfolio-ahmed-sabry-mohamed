# Task Description: Employee Performance Analysis

## Project Objective:
The goal is to analyze Human Resources (HR) data for a hypothetical company to understand the factors influencing employee performance. This includes correlating performance scores with attendance rates, training sessions, and salaries across various departments.

## Requirements:
1. **Data Organization:** Integrating multiple CSV files (Employees, Departments, Attendance, Evaluations, Trainings).
2. **Descriptive Analysis:** Calculating average salaries and employee distribution by gender and department.
3. **Performance Analysis:** Comparing performance evaluation scores with attendance and training data.
4. **Data Visualization:** Building an interactive dashboard using Tableau to present the findings clearly.

## Expected Deliverables:
- A relational database/integrated dataset.
- A visual report highlighting top-performing departments and the most committed employees.
