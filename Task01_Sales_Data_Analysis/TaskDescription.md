## Description
This task involves analyzing a dataset of sales records for the Egyptian market. The goal is to transform raw data into a clean format and generate visual insights to support decision-making.

## Objectives
- **Data Cleaning:** Transforming raw data into a structured format by handling missing values and inconsistencies.
- **Data Aggregation:** Using Excel Pivot Tables to summarize sales figures by region, product, and time period.
- **Visualization:** Creating a dynamic Excel Dashboard to display Key Performance Indicators (KPIs).

## Tools Used
- **Microsoft Excel:** Data cleaning, Pivot Tables, Charts, and Dashboard creation.

## Key Features
- Comparative analysis of sales performance across different regions in Egypt.
- Visual representation of sales trends over time.
